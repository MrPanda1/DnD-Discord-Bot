# DnD Discord Bot
This is a discord bot created to help speed up our virtual DnD sessions.

## Note
The current prefix for the bot is **?**.

## Current Commands
- Character
    - **create** *name* *race* *subrace*
    - **edit** *name* *race* *subrace*
    - **display** *(optional) name*
- Dice
    - **roll** *<# of dice>*d*<# of sides>*
